# Personal Expense Tracker

## Description

A console-based Personal Expense Tracker developed in Python using Object-Oriented Programming.

## Features

- Add Expense
- View Expenses
- Search Expense by ID
- Update Expense
- Delete Expense
- View Summary
- Category-wise Breakdown
- Auto Generated IDs
- Auto Date
- CSV File Storage

## Files

```
expense.py
expense_tracker.py
utils.py
main.py
expenses.csv
```

## How to Run

```bash
python main.py
```

## Data Storage

All records are automatically stored in

```
expenses.csv
```

and reloaded every time the application starts.

## Python Version

Python 3.10+
